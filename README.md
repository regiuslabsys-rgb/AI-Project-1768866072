"# PhoneStore - Лендинг для продажи телефонов\n\n## Описание\n\nЭтот проект представляет собой одностраничный лендинг для продажи телефонов. Лендинг содержит следующие разделы:\n\n*   Шапка (Header) с логотипом и навигационным меню.\n*   Главный экран (Hero section) с заголовком, описанием и призывом к действию.\n*   Каталог телефонов (Phones catalog) со списком товаров.\n*   О нас (About Us) с информацией о компании.\n*   Контакты (Contacts) с формой обратной связи и контактной информацией.\n*   Подвал (Footer) с копирайтом и ссылками.\n\n## Технологии\n\n*   HTML\n*   JavaScript (чистый JS)\n*   Tailwind CSS\n\n## Функциональность\n\n*   Валидация формы обратной связи (email, телефон).\n*   Обработка кликов по кнопкам \"Купить\" (вывод модального окна).\n*   Плавная прокрутка по якорям.\n*   Адаптивная верстка.\n\n## Установка и запуск\n\n1.  Клонируйте репозиторий.\n2.  Откройте файл `index.html` в браузере.\n\n## Структура проекта\n\n*   `index.html` - Основной файл с разметкой страницы.\n*   `script.js` - Файл с JavaScript-кодом.\n*   `style.css` - Файл со стилями CSS (Tailwind CSS).\n*   `phone.jpg, phone1.jpg, phone2.jpg, phone3.jpg, office.jpg` - Изображения для сайта.\n\n## Использование\n\nПросто откройте `index.html` в браузере.\n\n## Лицензия\n\nMIT\n"
